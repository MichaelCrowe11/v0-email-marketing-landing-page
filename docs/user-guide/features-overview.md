# Features Overview

Comprehensive guide to all Crowe Logic AI features and capabilities.

## AI Chat Assistant

### Overview
Engage in natural conversations with an AI trained on 20+ years of mycology expertise from Southwest Mushrooms.

### Key Capabilities
- **Expert Advice**: Get cultivation guidance based on proven techniques
- **Problem Solving**: Troubleshoot contamination, growth issues, and environmental problems
- **Species Information**: Learn about different mushroom species and their requirements
- **Technique Guidance**: Step-by-step instructions for cultivation processes

### How to Use
1. Click **New Chat** or navigate to the Chat section
2. Type your question or describe your situation
3. Add context like substrate type, environmental conditions, or growth stage
4. Upload images if relevant to your question
5. Review the AI's response with detailed reasoning (Premium feature)

### Tips for Best Results
- Be specific about your setup and conditions
- Include relevant details like temperature, humidity, and timeline
- Ask follow-up questions to clarify recommendations
- Reference previous conversations for continuity

### Premium Features
- **Unlimited Messages**: No monthly limits
- **Visible Reasoning Summary (VRS)**: See how the AI reaches conclusions
- **Priority Processing**: Faster response times
- **Conversation History**: Access all past chats

## Visual Analysis

### Overview
Upload images for AI-powered analysis of contamination, species identification, and growth assessment.

### Analysis Types

#### Contamination Detection
- Identifies bacterial, fungal, and mold contamination
- Provides confidence percentages for each finding
- Suggests remediation strategies
- Shows annotated images highlighting problem areas

#### Species Identification
- Identifies mushroom species from photos
- Provides morphological details
- Suggests similar species for comparison
- Links to detailed species information

#### Growth Stage Assessment
- Evaluates mycelium colonization progress
- Identifies pinning and fruiting stages
- Suggests optimal harvest timing
- Tracks development over time

### How to Use
1. Navigate to **Visual Analysis** or upload in chat
2. Select your image (JPG, PNG, WebP supported)
3. Choose analysis type or let AI auto-detect
4. Review annotated results with evidence ledger
5. Save analysis to your project for tracking

### Image Guidelines
- **Lighting**: Use natural light or diffused flash
- **Focus**: Ensure image is sharp and clear
- **Distance**: Close enough to show detail, far enough for context
- **Angle**: Multiple angles help for complex issues
- **Size Reference**: Include a ruler or known object when possible

### Understanding Results

#### Confidence Levels
- **90-100%**: Very high confidence, reliable identification
- **70-89%**: High confidence, likely accurate
- **50-69%**: Moderate confidence, verify with additional sources
- **Below 50%**: Low confidence, requires expert verification

#### Evidence Ledger
Each analysis includes supporting evidence:
- Visual markers and annotations
- Morphological characteristics observed
- Comparison to known examples
- References to knowledge base articles

## Project Management

### Overview
Track your cultivation projects from inoculation to harvest with detailed logging and photo documentation.

### Project Features

#### Project Creation
- Name and describe your project
- Set substrate type and species
- Define environmental parameters
- Set expected timeline and milestones

#### Observation Logging
- Daily or periodic observations
- Environmental data (temp, humidity, CO2)
- Photo documentation with timestamps
- Notes on changes or interventions

#### Progress Tracking
- Visual timeline of project stages
- Milestone completion tracking
- Yield and success metrics
- Comparison to previous batches

#### Analysis Integration
- Link visual analyses to projects
- Track contamination incidents
- Monitor growth progression
- Generate project reports

### How to Use
1. Navigate to **Projects** and click **New Project**
2. Fill in project details and parameters
3. Log observations regularly
4. Upload photos to document progress
5. Use AI chat to troubleshoot issues
6. Review project history and metrics

### Best Practices
- Log observations consistently (daily for critical stages)
- Take photos at the same time each day
- Record all environmental changes
- Note any interventions or adjustments
- Compare successful projects to identify patterns
