# Keyboard Shortcuts Guide

Crowe Logic AI provides keyboard shortcuts to help you navigate and use the platform more efficiently.

## Global Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/⌘ + K` | Open Search | Quickly search across the platform |
| `Ctrl/⌘ + T` | Toggle Theme | Switch between light and dark mode |
| `Ctrl/⌘ + N` | Focus Navigation | Jump to the navigation sidebar |
| `Ctrl/⌘ + S` | Skip to Content | Skip navigation and go to main content |
| `Shift + ?` | Show Help | Display this keyboard shortcuts dialog |

## Navigation Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Next Element | Move focus to the next interactive element |
| `Shift + Tab` | Previous Element | Move focus to the previous interactive element |
| `Arrow Keys` | Menu Navigation | Navigate through menu items when a menu is open |
| `Enter` | Activate | Activate the currently focused element |
| `Esc` | Close/Cancel | Close dialogs, menus, or cancel current action |

## Chat Interface Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl/⌘ + Enter` | Send Message | Send your message to the AI |
| `Ctrl/⌘ + /` | Focus Input | Jump to the chat input field |
| `↑` (in input) | Previous Message | Recall your previous message for editing |

## Accessibility Features

### Screen Reader Support
- All interactive elements have proper ARIA labels
- Dynamic content updates are announced via ARIA live regions
- Semantic HTML structure for easy navigation

### Keyboard Navigation
- All functionality is accessible via keyboard
- Visible focus indicators show your current position
- Logical tab order follows visual layout

### Reduced Motion
- Enable reduced motion in your system preferences
- Or toggle in Settings → Accessibility → Reduce Motion
- Animations will be minimized or disabled

## Tips for Efficient Use

1. **Learn the Search Shortcut**: `Ctrl/⌘ + K` is your fastest way to find anything
2. **Use Tab Navigation**: Navigate forms and interfaces without touching your mouse
3. **Enable Reduced Motion**: If animations are distracting, turn them off
4. **Customize Your Experience**: Visit Settings to adjust font size, contrast, and more

## Platform-Specific Notes

### Windows/Linux
- Use `Ctrl` for all shortcuts
- Example: `Ctrl + K` for search

### macOS
- Use `⌘` (Command) for all shortcuts
- Example: `⌘ + K` for search

## Need Help?

Press `Shift + ?` at any time to see this shortcuts guide, or visit our [Help Center](../docs) for more information.
