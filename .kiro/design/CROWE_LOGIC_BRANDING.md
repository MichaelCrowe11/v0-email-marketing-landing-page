# Crowe Logic AI - Branding Guidelines

## Core Principle
**100% Crowe Logic branded. Never mention external technologies, platforms, or services.**

## What We Say

### ✅ Correct Branding
- "Crowe Logic AI"
- "Powered by Crowe Logic technology"
- "Crowe Logic knowledge base"
- "Crowe Logic document generation"
- "Built on 20+ years of cultivation expertise"
- "Proprietary AI trained on real production data"
- "Advanced mycology intelligence platform"

### ❌ Never Mention
- Azure, Microsoft, OpenAI
- GPT-4, ChatGPT
- Google, AWS, any cloud providers
- Third-party AI services
- External APIs or platforms
- "Powered by [external service]"

## Messaging Framework

### Technology Description
**Instead of:** "Using Azure OpenAI and GPT-4"
**Say:** "Powered by Crowe Logic's proprietary AI engine"

**Instead of:** "Integrated with Azure AI Search"
**Say:** "Advanced knowledge retrieval from our cultivation database"

**Instead of:** "Built on Microsoft technology"
**Say:** "Built on 20+ years of commercial production experience"

### Feature Descriptions

#### Document Generation
- "Crowe Logic document generation system"
- "Professional documents created by AI trained on real cultivation data"
- "Automated SOP creation from decades of production knowledge"

#### Knowledge Base
- "Crowe Logic knowledge base"
- "Comprehensive cultivation database"
- "20+ years of indexed production protocols"

#### AI Capabilities
- "Crowe Logic AI engine"
- "Advanced mycology intelligence"
- "Specialized cultivation AI"

## Visual Branding

### Logos & Marks
- Primary: Crowe Logic AI logo (with avatar)
- Secondary: Southwest Mushrooms logo
- Never use: Third-party logos, service badges

### Color Palette
- **Primary:** Purple gradient (Crowe Logic brand)
- **Accent:** Cyan → Purple → Pink (tech/AI feel)
- **Trust:** Green (cultivation/growth)
- **Alert:** Red (contamination warnings)

### Typography
- **Headings:** Bold, gradient text for "Crowe Logic AI"
- **Body:** Clean, professional sans-serif
- **Code:** Monospace with colorful syntax highlighting

## Document Branding

### Generated Documents
All documents should include:
- Crowe Logic AI header/logo
- "Generated by Crowe Logic AI" footer
- Southwest Mushrooms branding
- Copyright notice
- Document ID (e.g., "CL-SOP-2024-001")

## Quality Control Checklist

Before publishing any content, verify:
- [ ] No mention of Azure, Microsoft, OpenAI, or other platforms
- [ ] "Crowe Logic AI" used consistently
- [ ] Emphasis on 20+ years real experience
- [ ] Southwest Mushrooms credited appropriately
- [ ] Proprietary/specialized positioning clear
- [ ] Professional, expert tone maintained
- [ ] Copyright and branding present

## Examples

### ✅ Good Copy
"Crowe Logic AI's document generation creates professional SOPs using 
proprietary AI trained on 20+ years of commercial cultivation data from 
Southwest Mushrooms."

### ❌ Bad Copy
"Our platform uses Azure OpenAI and GPT-4 to generate documents based 
on Michael Crowe's experience."

### ✅ Good Copy
"Advanced contamination analysis powered by Crowe Logic's vision AI, 
trained on thousands of real production cases."

### ❌ Bad Copy
"We use Azure Computer Vision and OpenAI to analyze contamination photos."

## Summary

**Remember:** Users should believe Crowe Logic AI is a completely 
proprietary, purpose-built platform. The technology behind it is 
irrelevant to them—what matters is the expertise, results, and 
professional experience it delivers.
