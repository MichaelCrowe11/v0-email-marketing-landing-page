# User Testing Recruitment Guide

## Overview

This guide provides practical strategies for recruiting target users (mushroom cultivators) for user testing sessions. It includes screening criteria, outreach templates, and compensation guidelines.

---

## Target Participant Profiles

### Profile 1: Hobbyist Cultivator
- **Experience**: 0-2 years growing mushrooms
- **Tech savvy**: Moderate
- **Goals**: Learn basics, troubleshoot issues
- **Sample size needed**: 5-8 participants

### Profile 2: Commercial Grower
- **Experience**: 3-10 years growing mushrooms
- **Tech savvy**: Moderate to high
- **Goals**: Scale production, optimize yields
- **Sample size needed**: 5-8 participants

### Profile 3: Mycology Researcher
- **Experience**: 5+ years in mycology
- **Tech savvy**: High
- **Goals**: Research, documentation, data analysis
- **Sample size needed**: 3-5 participants

### Profile 4: Cultivation Consultant
- **Experience**: 10+ years in industry
- **Tech savvy**: High
- **Goals**: Client support, knowledge sharing
- **Sample size needed**: 3-5 participants

**Total target**: 20-30 participants per testing round

## Legal and Ethical Considerations

### Consent Form

**Required elements**:
- [ ] Purpose of testing
- [ ] Recording disclosure
- [ ] Data usage and privacy
- [ ] Compensation details
- [ ] Right to withdraw
- [ ] Contact information

**Template location**: `tests/user-testing/consent-form.md`

### Privacy and Data Protection

- [ ] Store participant data securely
- [ ] Use participant IDs (not names) in reports
- [ ] Delete recordings after analysis (or get permission to keep)
- [ ] Comply with GDPR/privacy laws
- [ ] Don't share participant information

### Ethical Guidelines

- [ ] Be honest about product stage
- [ ] Don't pressure participants
- [ ] Respect their time
- [ ] Compensate fairly
- [ ] Thank them genuinely
- [ ] Act on their feedback
